# crop-disease-detection-ml
An end-to-end machine learning application that detects plant leaf diseases from images using a trained model, with an interactive Streamlit interface, confidence visualization, and remedy recommendations.

# 🌿 Crop Disease Detection using Machine Learning

## 📌 Overview
This project is a machine learning-based web application that detects plant leaf diseases from images. It helps in early identification of crop diseases, reducing crop loss and improving agricultural productivity.

---

## 🚀 Features
- Upload leaf images for analysis
- Detect whether the leaf is healthy or diseased
- Display prediction confidence score
- Identify disease type
- Provide remedy suggestions
- Interactive and user-friendly interface using Streamlit
